离线办公助手 - 完整功能列表 (适用于 GitHub 开源项目)
一、核心文档处理
富文本编辑器

支持粗体、斜体、下划线、删除线、上标/下标

字体、字号、颜色、背景色、对齐方式、列表（有序/无序）、缩进

表格插入与编辑、图片插入（本地/粘贴）、超链接

页眉/页脚、页码、分页符、页面边距设置

保存格式：.rtf, .docx (通过libreoffice或自研转换), .odt

Markdown 编辑器

实时渲染预览、语法高亮、数学公式 (KaTeX/MathJax)

支持 Mermaid 流程图、时序图、甘特图

导出为 PDF, HTML, 图片 (PNG/JPG)

支持 Front Matter (YAML 元数据)

纯文本编辑器

大文件打开（>100MB 流畅）

编码自动检测 (UTF-8/GBK/ASCII等)

行号、括号匹配、自动换行、查找替换（正则支持）

电子表格

基本公式：SUM, AVERAGE, COUNT, MAX, MIN, IF, VLOOKUP 等

单元格格式：数字、货币、日期、百分比、文本

排序、筛选、条件格式、数据验证

多工作表、冻结窗格、图表生成（柱状、折线、饼图，基于ECharts）

导入/导出 .xlsx, .csv, .ods

演示文稿

幻灯片编辑：文本框、图片、形状、表格、图表

过渡动画、对象动画（飞入、淡出等）

母版视图、主题模板（预置3-5套）

导出为 PDF, 图片, 支持打印讲义

格式：.pptx (只读基本支持), 或自研 .odp

二、PDF 工具集
PDF 阅读器

目录书签、缩略图导航、文本搜索

缩放、旋转、双页/单页模式

夜间模式、高亮文本、下划线、删除线、手绘涂鸦

表单填写（文本框、复选框）

PDF 编辑器

添加/删除/旋转页面、提取页面、拆分合并

添加文本、图片、形状、签名（手写/图片）

加密/解密（密码保护、权限设置）

将 PDF 转换为图片（每页为 PNG/JPG）

PDF 转换器

PDF 转 Word (.docx)、Excel (.xlsx)、HTML、TXT

图片转 PDF（多图合并在同一文件，可排序）

三、文件与格式转换
批量文件转换

图片格式：jpg, png, webp, bmp, tiff 互转

文档转换：docx ↔ odt, xlsx ↔ csv, pptx ↔ odp

文本编码转换（GBK ↔ UTF-8 等）

压缩与解压缩

支持 zip, 7z, tar, gz, rar (只读)

分卷压缩、加密压缩、添加注释

解压到指定文件夹，支持拖放

OCR 文字识别（离线）

基于 Tesseract 或 PaddleOCR 引擎

支持中英文混合、自动版面分析

从图片/PDF 识别文字后输出为 TXT 或 Word

可校正识别结果、区域选择识别

四、辅助办公工具
剪贴板管理

历史记录（文本、图片、文件路径，最多 200 条）

置顶常用条目、合并粘贴、支持全文搜索

可导出剪贴板记录为 txt 或 json

截图与屏幕录制

矩形/任意形状截图、自动识别窗口

截图后标注：箭头、矩形、圆形、文字、马赛克

屏幕录制（区域/全屏）输出为 mp4 或 gif

录音转文字（离线）

基于 Whisper.cpp 等轻量模型

支持实时麦克风录入或导入音频文件 (mp3/wav)

自动添加标点、时间戳导出为 srt/vtt 字幕或纯文本

待办事项与提醒

列表式待办（支持子任务、优先级、标签）

定时提醒（桌面通知，可设置重复）

集成番茄钟专注计时（25分钟/5分钟）

数据本地存储 (sqlite) 或同步 (WebDAV 可选)

五、数据处理与开发辅助
JSON / XML 格式化

树形查看、代码折叠、语法高亮

验证、压缩、转义/反转义

JSON 与 XML 互转、JSON 与 YAML 互转

正则表达式测试器

实时匹配高亮、捕获组显示

常用正则库（邮箱、URL、手机号等）

可针对本地文件内容进行批量搜索替换

代码片段管理

支持多种语言高亮 (JavaScript, Python, SQL, Shell 等)

分类管理、快捷插入到当前编辑器

支持变量占位符（如 $date$）自动替换

流程图与思维导图

基于 Draw.io 离线版或 Excalidraw

导出为 png/svg、嵌入到 Markdown

支持拖拽形状、连接线、样式库

六、系统与实用工具
文件搜索

按文件名/内容全文搜索（支持正则）

索引指定文件夹（后台增量更新）

搜索结果可复制、移动、删除

批量重命名

支持编号、替换、插入、删除、正则匹配

大小写转换、扩展名修改

实时预览新文件名

加密解密工具

对称加密：AES-256 (文件/文本)

非对称加密：RSA (生成密钥对)

文件粉碎（覆写多次）

图片处理

裁剪、旋转、调整大小、滤镜（灰度、锐化等）

格式转换、压缩质量调节

添加水印（文字或图片）

词典与翻译（离线）

内置英汉/汉英词库（例如 StarDict 格式）

划词翻译、取词悬停

支持简单句子翻译（本地模型如 OPUS-MT 可插拔）

计算器

基本运算、科学计算（三角函数、对数）

表达式计算（支持括号、函数）

单位换算：长度、重量、面积、体积、温度、货币（需要手动更新汇率）

七、笔记与个人信息管理
树形笔记

无限层级目录、标签系统、全文搜索

支持插入待办 checkbox、表格、代码块

数据存储为本地 .md 或 .json，可导出 zip

日历与日程

月/周/日视图、事件添加（标题、地点、描述）

重复事件、闹铃提醒、颜色标签

导入/导出 .ics (iCalendar)

联系人管理

姓名、电话、邮箱、地址、生日字段

分组、快速搜索、导出 vCard

便签（桌面便签）

浮动贴纸、置顶、半透明背景

支持拖拽、自动保存、多颜色区分

八、自动化与宏
宏录制与回放

录制键盘鼠标操作（保存为 .macro）

支持循环执行、延迟设置

可绑定快捷键触发

批量文件处理工作流

可视化拖拽步骤：重命名 → 转换格式 → 压缩 → 移动

保存工作流为模板，一键运行

九、界面与可扩展性
主题系统

亮色/暗色/跟随系统

自定义背景色/强调色

插件机制

基于 JavaScript 或 Lua 的插件 API

支持添加新功能、自定义工具栏按钮

多语言界面

中文简体、英文（通过 gettext 或 i18next）

数据备份与恢复

定时备份整个工作目录到指定路径

导出用户设置、一键恢复

十、性能与离线保障
所有功能完全离线可用，无需互联网

使用 SQLite 存储本地数据（笔记、联系人、剪贴板历史等）

支持大型文档分页加载 (虚拟滚动)

CPU/内存占用优化，闲置时释放资源

支持便携模式（数据存储在程序目录）

十一、交付要求（GitHub 项目）
采用 Electron + Vue/React 跨平台桌面框架

后端使用 Node.js + Rust (Tauri) 或 Go 实现高性能转换

开放源代码 (MIT 或 GPLv3 许可证)

提供 Windows (.exe)，macOS (.dmg)，Linux (.AppImage) 预编译包

文档包含：功能说明、编译指南、插件开发教程

单元测试覆盖率 > 70%，CI 自动构建
